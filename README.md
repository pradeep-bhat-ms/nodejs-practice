# Node.js Daily Learning Journey 🚀

Welcome to my Node.js learning repository!

This repository contains my daily practice programs and examples while learning Node.js fundamentals. The goal is to build a strong foundation in backend development by understanding Node.js core modules, asynchronous programming, event-driven architecture, and server-side development.

* CommonJS Modules (`require`, `module.exports`)

* Importing and exporting functions and objects

* Reading files
* Writing files
* Updating files
* Appending data to files
* Synchronous file operations

### Day 3 - File System Promises & Async/Await

* `fs/promises`

* Creating HTTP servers

## 🚀 How to Run

Clone the repository:

```bash
git clone https://github.com/pradeep-bhat-ms/nodejs-practice.git
```

Move to the project directory:

```bash
cd nodejs-practice
```

Run any JavaScript file:

```bash
node filename.js
```

Example:

```bash
node server.js
```

Or with Nodemon:

```bash
nodemon filename.js
```
